## PM2.5
训练集有240天，每天18个变量，24个小时（后15个小时作为结果），如果把所有数据全部运用可能会出现过拟合的情况。

检索论文后得知PM2.5指数与氮氧化物、SO<sub>2</sub>有关<sup>[1]</sup>，故选取“NO"、"NO<sub>2</sub>"、"NO<sub>x</sub>"、"PM2.5"、"SO<sub>2</sub>"作为训练数据。
每天从0时至14时每次向后取9个小时（含开头）作为一组训练数据，共15组。再从240天的数据中随机抽取192天作为Training，剩余48天作为Validation。

当训练至平均Loss小于50左右时，Loss收敛，此时在validation上的拟合效果已非常准确。训练出的bias，theta和在validation上的拟合图（随机验证了三次）已上传。
